# Wisdom Woods 🌳

Wisdom Woods is a playful educational web app built for kids aged 8–12. Explorers create their own profile, pick a guide, and venture through a series of interactive learning games — earning XP and sharpening their skills along the way.

🔗 [Visit Wisdom Woods](https://dipdagod.github.io/Wisdom-Woods/main.html)

## Features

* 🧠 **Brain Puzzles & Riddles** – Logic challenges to get the gears turning
* ➗ **Math Quest** – An adventure through numbers and problem-solving
* 🃏 **Memory Flip Game** – Classic card-matching for memory practice
* 🌴 **Word Jungle** – Vocabulary building in a jungle setting
* ✅ **True or False Quiz** – Quick-fire fact checking
* ✍️ **Sentence Mix** – Rearrange words to build correct sentences
* 🧩 **Fill the Gap** – Complete the missing pieces in a sentence
* ⚡ **Speed Maths** – Race the clock with rapid-fire calculations
* 🔬 **Science Sort** – Sort and classify science concepts
* 🌍 **World Explorer** – Discover geography and world facts
* 🤖 **Grammar AI Tool** – Get instant, AI-powered grammar feedback
